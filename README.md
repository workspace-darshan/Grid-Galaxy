## ✨ Features

- **🎬 GSAP-Powered Animations** — Buttery-smooth entrance animations, scroll triggers, and timeline sequences using GreenSock's industry-leading animation platform
- **🧩 Grid Showcase** — Interactive, visually rich demonstrations of CSS Grid concepts and layouts
- **⚡ Blazing Fast** — Built with Vite 7 for near-instant dev startup and optimized production builds
- **🔷 Fully Typed** — Written in TypeScript for robust, maintainable code
- **🎨 Tailwind CSS v4** — Utility-first styling with the latest Tailwind architecture
- **📦 Single-File Build** — Uses `vite-plugin-singlefile` to bundle everything into one portable HTML file
- **🌐 Zero Dependencies at Runtime** — No backend, no API calls — pure frontend excellence
- **📱 Responsive Design** — Looks great on all screen sizes

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) `v18+`
- [npm](https://npmjs.com/) or your preferred package manager

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/workspace-darshan/Grid-Galaxy.git

# 2. Navigate into the project
cd Grid-Galaxy

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Your app will be running at **http://localhost:5173** 🎉

## 🎨 Design Philosophy

Grid Galaxy is more than a code demo — it's a **design statement**.

- **Typography**: A deliberate pairing of `Space Grotesk` (modern, technical) with `Playfair Display` (elegant, editorial) creates visual tension that feels intentional and premium.
- **Motion First**: Every element is animated with purpose. GSAP handles complex timelines so transitions feel natural, not jarring.
- **Grid as Art**: CSS Grid isn't just a layout tool here — it's the artistic medium. The site treats grid structures as compositions worth exploring visually.

## 🌍 Deployment

This project is deployed on **Netlify** with continuous deployment from the `main` branch.

To deploy your own fork:

1. Push your code to GitHub
2. Connect your repo to [Netlify](https://netlify.com/)
3. Set the build command to `npm run build`
4. Set the publish directory to `dist`

That's it — your own Grid Galaxy is live! 🚀
